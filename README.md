# 📚 reading_material — 知識インプット用 読み物ライブラリ

プログラミングに偏りがちな知識を、**分野横断の短編エッセイ**で広げるためのライブラリです。
1本3〜5分で読める気軽な読み物に、末尾で **「💡 アプリアイデアの種」** を添えています。
知らない分野の視点をインプットして、アプリ開発の着想につなげましょう。

> 運用ルール・記事フォーマットは [CLAUDE.md](./CLAUDE.md) を参照。

## 読み方

- 気になったタイトルから、どこでも自由に読んでOK。
- 各記事の末尾に、その知識から着想できるアプリ案が数個ついています。
- 追加の読み物が欲しくなったら「〇〇の分野の読み物を追加して」と伝えてください（サブエージェント並列で量産します）。
