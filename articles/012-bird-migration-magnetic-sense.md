# 渡り鳥はなぜ迷わない？体内の量子コンパス

毎年秋になると、無数の鳥たちが北の繁殖地から南の越冬地へ、何千キロもの旅に出る。北極圏で夏を過ごし南極付近まで飛ぶキョクアジサシは、往復でおよそ7万キロ、地球一周半以上を毎年移動する。地図もGPSも持たず、初めての旅路でも彼らはめったに迷わない。いったいどうやって進むべき方角を知るのだろう。

鳥たちはいくつもの「道しるべ」を組み合わせている。昼は太陽の位置、夜は星の並び、地形や海岸線、匂いまで手がかりにする。だが曇りで太陽も星も見えない日でも、彼らは正しい方角を保てる。ここで登場するのが、いちばん不思議な能力——「地磁気を感じる力（磁気感覚）」だ。

地球は巨大な磁石で、方位磁針のN極が北を指すように、地表には目に見えない磁力の線が走っている。渡り鳥はこの地磁気を「感じ取り」、体内コンパスとして使っていることが実験で分かっている。人工的に磁場の向きを変えた部屋に鳥を入れると、彼らはだまされて違う方角へ飛ぼうとするのだ。

問題は「どうやって磁気を感じているのか」だ。ここ数十年でもっとも有力とされる仮説が、驚くほど現代的だ。鳥の目の網膜には「クリプトクロム」というタンパク質があり、青い光が当たると、その中で電子が特殊なペア(ラジカル対)を作る。このペアの状態が、周囲の地磁気の向きによってごくわずかに変化する——そしてこの変化には、原子より小さな世界を支配する「量子力学」の効果が関わっているというのだ。つまり鳥は、目を通して磁場を「模様」や「明るさのムラ」のような視覚情報として見ているのかもしれない。私たちには想像もつかないが、彼らの目には北の方角がうっすら光って見えている可能性がある。

さらに近年の研究では、ヨーロッパコマドリのクリプトクロムを実験室で調べたところ、確かに磁場に反応する性質を持つことが確認された。生き物の体が、実験装置のような繊細さで量子の効果を利用している——「量子生物学」という新しい学問分野が、ここから広がりつつある。

ただし、まだ謎は多い。目のコンパスだけでなく、くちばしの周辺に磁鉄鉱の粒があってそこでも磁気を感じている、という説もあり、鳥は複数のセンサーを使い分けている可能性が高い。方角(コンパス)は分かっても「今どこにいるか(地図)」をどう把握しているのかは、いまだに完全には解明されていない。

私たちがスマホの地図に頼って歩く一方で、渡り鳥は体の中に太陽と星と磁石を内蔵し、量子のゆらぎまで動員して大空を旅している。頭上を渡っていく鳥の群れは、生命が進化の中で作り上げた、世界最高峰のナビゲーション装置なのだ。
