# Save the Cat：15のビートで物語を組む

三幕構成が物語の「大骨格」なら、それをさらに細かい設計図に落とし込んだのが、脚本家ブレイク・スナイダーの**「Save the Cat!(セーブ・ザ・キャット)」**だ。彼は大ヒット映画を分析し、面白い脚本には**15の「ビート(拍)」**が、ほぼ決まったページ数の位置に並んでいることを発見した。ビートとは、物語を前に進める節目の出来事のこと。この15拍を並べた「ビートシート」は、いまやハリウッド脚本家の必携ツールになっている。

タイトルの由来がまず面白い。「Save the Cat=猫を助ける」とは、**物語の冒頭で、主人公が何か良いことをする場面**を指す。木に登って降りられなくなった猫を助ける、困っている人に手を貸す――そんな小さな善行だ。なぜ必要か。観客は、これから2時間つき合う主人公を「応援したい」と思えなければ物語に乗れない。だから序盤に「この人、いいやつだな」と思わせる一瞬を仕込む。たとえ主人公が泥棒でも殺し屋でも、犬を助ける一場面があれば観客は味方になる。人の心を掴む、驚くほど実用的なテクニックだ。

15ビートの主なものを挙げよう。冒頭の「オープニング・イメージ」で世界観を提示し、「テーマの提示」で作品が問う問いをさりげなく置く。「セットアップ」で日常を描き、「きっかけ」で事件が起き、「悩みのとき」で主人公が迷う。「第一ターニングポイント」で非日常へ飛び込み、中盤の「ミッドポイント」で状況が反転。「迫りくる悪い奴ら」でじわじわ追い詰められ、「全てを失って」どん底に落ち、「心の暗闇」で絶望する。そこから「第二ターニングポイント」で立ち直る糸口を掴み、「フィナーレ」で決着、「ファイナル・イメージ」で冒頭と対になる変化後の姿を見せる。

『ダイ・ハード』でも『ショーシャンクの空に』でも、この15拍はきれいに見つかる。特に効くのが冒頭と結末の「イメージの対」だ。始まりのカットと終わりのカットが呼応し、「主人公はここまで変わった」を無言で語る。脚本の完成度は、この対比の美しさに表れる。

Save the Cat の思想の核心は、**「型は自由を奪うのではなく、自由を支える」**という点にある。15の空欄が用意されていると、作り手は「次に何を書けばいいか」で悩まず、「その拍に何を入れると面白いか」に集中できる。料理でいえばレシピの手順が決まっているから、素材と味付けの創意に頭を使える。制約はしばしば、創造性の敵ではなく親友なのだ。

## 💡 アプリアイデアの種

- 案1: 15ビートのテンプレートに沿って物語を組み立てるプロット作成アプリ。各ビートの理想ページ位置を示し、埋まっていない拍をハイライト。初心者でも構成の穴に気づける。
- 案2: 「猫を助ける」発想を応用した第一印象デザイン支援。アプリのオンボーディングやプロフィール作成で、ユーザーが最初に「好感」を得る一手を設計するUXガイド。
- 案3: 映画・ドラマを観ながら15ビートの位置をタップ記録できる分析アプリ。作品ごとのビート配置を可視化・比較し、脚本の型を体で覚える学習ツール。
