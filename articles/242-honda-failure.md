# 「99%の失敗に支えられた1%」——本田宗一郎が失敗を愛した理由

「私がやってきた仕事で成功したのは、全体のわずか1%にすぎない。99%は失敗の連続だった」——本田技研工業(HONDA)の創業者・本田宗一郎は、こうした趣旨の言葉をたびたび口にした。「成功とは、99%の失敗に支えられた1%だ」というかたちで語り継がれている。世界的な大企業を築いた人物が、自らの人生をほぼ失敗だったと言い切るのは、なかなか痛快である。

本田宗一郎は1906年、静岡の鍛冶屋の長男に生まれた。学歴は高等小学校卒。少年時代から機械が好きで、自動車修理工場で丁稚として腕を磨いた。独立してピストンリングの製造に挑むが、当初は品質基準を満たせず、大量の不良品を出して苦しむ。技術が足りないと痛感した彼は、なんと三十代半ばで地元の工業高校に聴講生として通い、金属工学を学び直したという逸話も残る。プライドより実利をとる人だった。

戦後、彼は自転車に小さなエンジンを付けた乗り物から再スタートを切る。物資も燃料も乏しい時代、これが「安く走れる移動手段」として飛ぶように売れた。そこから二輪、そして四輪へ。技術者たちは何度もエンジンを壊し、レースで惨敗し、作り直しては挑んだ。名機と呼ばれるエンジンの裏には、数え切れない失敗作の山があった。

本田がユニークなのは、失敗を精神論で片づけず、**失敗こそが次への唯一の情報源だと考えた**点だ。「うまくいかなかった」という結果は、「この方法ではダメだ」という貴重なデータでもある。だから彼は挑戦して失敗した部下を頭ごなしに叱らなかったと言われる。むしろ「チャレンジして失敗を恐れることよりも、何もしないことを恐れろ」という趣旨のメッセージを社内文化として残した。失敗を罰する組織では、誰も新しいことに手を出さなくなる——それを彼は肌で知っていた。

この失敗許容の文化は、後年「失敗を表彰する」ユニークな社内賞という形にも表れたと語られることがある。最も果敢に挑んで最も大きく失敗した研究に賞を与える、という発想だ。もっとも、その具体的な運用については諸説あり、美談として脚色されている部分もあるかもしれない。ただ、そうしたエピソードが違和感なく本田の名と結びつくところに、この人の思想の一貫性がうかがえる。

1%の成功だけを見て「あの人は天才だ」と言うのはたやすい。だが本人が誇ったのは、その1%を支えた99%の失敗の厚みだった。転んだ回数こそが財産になる——挑戦を続ける人への、心強いお守りのような言葉である。

- 案2: チームで「最も果敢な失敗」を称え合う社内表彰アプリ。挑戦の数を可視化し、失敗を恐れない文化づくりを後押しする。
