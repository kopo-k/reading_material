# テトリス効果──脳は「やりかけ」を勝手に再生する

テトリスを長時間プレイしたあと、目を閉じるとまぶたの裏で落ちてくるブロックが見えたり、スーパーの棚を見て「この形ならあの隙間に入るな」と考えてしまったり──そんな経験はないだろうか。これは「テトリス効果」と呼ばれる、れっきとした心理現象だ。1994年に学術誌『Science』の記事でも取り上げられ、その後の実験でも繰り返し確認されている。

面白いのは、この効果が記憶に問題を抱えた人にも現れることだ。ハーバード大学の研究チームが、重い健忘症の患者たちにテトリスをプレイさせた実験がある。彼らは「昨日テトリスをやった」という事実そのものはまったく思い出せない。それなのに、夜になると「落ちてくるブロックの映像が見えた」と報告したのだ。出来事を覚える記憶（エピソード記憶）と、体で覚える記憶（手続き記憶）は別々の仕組みで動いている、という何よりの証拠である。

なぜ脳はこんなことをするのか。カギは「ツァイガルニク効果」にある。ロシアの心理学者ブルーマ・ツァイガルニクは、カフェのウェイターが会計前の注文は細かく覚えているのに、支払いが済んだ途端きれいさっぱり忘れることに気づいた。人間の脳は「完了したこと」より「未完了のこと」を強く覚え続けるのだ。テトリスは常に「まだ消せる列がある」「次のブロックをどう置くか」という未完了状態を作り出す。だから脳はゲームを離れても、その処理を勝手に続けてしまう。

この現象、じつは悪用も活用もできる。ネガティブな側では、つらい経験の映像が勝手にフラッシュバックする心的外傷(トラウマ)も、脳が「未処理の情報」を反芻している状態だと考えられている。ところが同じハーバードの研究者は、交通事故の直後にテトリスをプレイしてもらうと、その後のフラッシュバックが有意に減ることを見つけた。視覚的なゲームが脳の「映像を焼き付ける回路」を先に占領し、つらい記憶の定着を邪魔するらしい。

ポジティブな活用もある。難しい問題を解いている途中でわざと中断すると、脳は「未完了」として抱え続け、シャワーを浴びている時にふと答えがひらめく。作家が章の途中でペンを置くと、翌日すぐ続きを書き出せるのも同じ理屈だ。「キリのいいところまでやる」より「あえて中途半端でやめる」ほうが、再開のエンジンはかかりやすい。

私たちは自分の意思で考えていると思いがちだが、脳の相当な部分は「やりかけの仕事」に引きずられて勝手に回り続けている。今日あなたの頭に何度も浮かんでくるあの考えごとは、もしかすると「まだ終わっていませんよ」という脳からのしつこいリマインダーなのかもしれない。

## 💡 アプリアイデアの種

- 案1: ツァイガルニク効果を逆手に取った学習アプリ。問題をあえて「解ききる直前」で区切って通知し、再開率と定着率を高める。
- 案2: 嫌な出来事の直後にテトリス風の視覚パズルを一定時間プレイさせ、フラッシュバックの軽減を狙うメンタルケア・ツール。
- 案3: タスク管理アプリに「わざと中断」機能。作業を切りのいい所ではなく途中で止めるよう促し、翌日の再開速度を計測して見せる。
- 案4: 就寝前のプレイで見た「まぶたの裏の映像」を記録し、テトリス効果や夢の内容との関連を可視化するセルフ観察アプリ。
