# アイデアの発想：What if の魔法

物語のアイデアは、どこからやってくるのか。作家たちに聞くと、答えは驚くほど共通している。「もし〜だったら？」という、たった一つの問いだ。英語で言えば **"What if?"**。この短い呪文が、平凡な日常を物語のエンジンに変える。

たとえばSF作家のフィリップ・K・ディックは、こう問うた。「もし第二次世界大戦で枢軸国が勝っていたら？」。そこから『高い城の男』が生まれた。スティーヴン・キングの発想はもっと身近だ。彼はエッセイ集『書くことについて』で、アイデアは「二つの無関係なものを衝突させる」ことから生まれると書いている。『キャリー』は、「もし、いじめられている内気な少女に、実は超能力があったら？」という問いから始まった。高校時代の同級生の記憶と、念力という異物を衝突させたのだ。

面白いのは、What if が **二段構え** になっていることだ。まず大きな問い（もし恐竜を現代に蘇らせたら？）を立て、次に「だとしたら、何が起きる？」と因果を転がしていく。マイケル・クライトンの『ジュラシック・パーク』は、この転がしの連続で出来ている。恐竜を蘇らせる → 見世物にしたくなる → テーマパークを作る → 制御が効かなくなる。一つの「もし」が、次の「だとしたら」を呼ぶ。物語とは、問いのドミノ倒しなのだ。

ピクサーはこの手法を組織の作法にまで高めた。彼らの企画は、ほぼすべて What if で要約できる。「もしオモチャに感情があったら？（トイ・ストーリー）」「もし感情そのものがキャラクターだったら？（インサイド・ヘッド）」「もしネズミが一流シェフを夢見たら？（レミーのおいしいレストラン）」。共通するのは、**「ありえない前提」と「切実な感情」の掛け算** だ。突飛な設定だけでは空回りする。そこに「認められたい」「別れが怖い」といった人間の本音が乗ると、突然、物語が呼吸を始める。

この発想法の優しいところは、才能がいらないことだ。必要なのは観察と、ほんの少しのいたずら心。通勤電車で、隣の人がずっとスマホの同じ画面を見ている。ここで「もし、あの人が誰かからの返信を10年待っているとしたら？」と問うだけで、もう物語の入り口に立っている。日常のあらゆる場面に「もし」をかけてみる。信号待ち、コンビニのレジ、雨の匂い。世界は、問いをかけられるのを待っている素材の山だ。

コツは、答えを急がないこと。良い What if は、すぐに答えが出ない。むしろ「うわ、それどうなるんだろう」と自分がワクワクする問いこそが当たりだ。作家のアイデア帳は、完成した物語ではなく、この宙ぶらりんの問いで埋まっている。魔法の杖は、誰の手にもある。振り方を覚えるだけでいい。

## 💡 アプリアイデアの種

- 案1: 日常の写真を撮ると「もし〜だったら？」の問いを3つ自動生成してくれる発想トレーニングアプリ。物語のタネを毎日ストックできる。
- 案2: 「前提（ありえない設定）」×「感情（切実な本音）」をランダムに掛け合わせて企画の骨子を出すブレストツール。ピクサー式の掛け算を再現。
- 案3: 一つの What if から「だとしたら？」を連鎖入力していくと、因果のドミノを可視化してプロット骨格に変換してくれるアウトライン支援アプリ。
