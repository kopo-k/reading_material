# 「蝶のように舞い、蜂のように刺す」——名文句を作ったのは、実はアリではなかった

「Float like a butterfly, sting like a bee.（蝶のように舞い、蜂のように刺す）」。ボクシング史上最高のヒーロー、モハメド・アリを象徴する、あまりにも有名なフレーズだ。ヘビー級の巨漢でありながら、まるで軽量級のように華麗にステップを踏み、鋭いパンチで相手を仕留めるアリの闘い方を、これ以上ないほど的確に言い表している。多くの人はこれを、アリ本人が生み出した天才的なキャッチコピーだと信じている。

ところが、その真相は少し違う。このフレーズを考えたのは、アリの陣営を支えた側近、ドリュー・"バンディーニ"・ブラウンだったと言われている。バンディーニはトレーナーではないが、アリのそばに寄り添い、士気を高め、言葉で場を沸かせる相棒のような存在だった。試合前の記者会見でアリを盛り上げるための煽り文句、いわば二人三脚で作り上げたパフォーマンスの一部として、この名フレーズは生まれた。アリはそれを自分のリズムに乗せ、韻を踏んだ挑発詩の一節として世界に叩きつけ、自分の代名詞にしてしまったのだ。言葉を「作った」のはバンディーニでも、それを「伝説にした」のは間違いなくアリだった。

アリという人物は、リングの外でこそ真価を発揮した。本名はカシアス・クレイ。ボクシングで世界の頂点に立った後、イスラム教に改宗してモハメド・アリと名を改める。そして1960年代、ベトナム戦争のさなか、彼は徴兵を拒否した。「自分には、遠い国の人々と争う理由がない」という信念からだった。この決断の代償は大きかった。彼は王座を剥奪され、ボクサーとしてもっとも脂の乗った数年間を、リングに立てないまま奪われてしまう。

だが、アリは折れなかった。有罪判決と闘い、信念を貫き、やがて最高裁で逆転の判断を勝ち取る。そしてリングに復帰し、再び世界王者へと返り咲いた。「蝶のように舞い、蜂のように刺す」——この言葉は、実はボクシングのスタイルだけを指しているのではない。権力に対しては軽やかにかわしながら、言うべきことは鋭く刺す。彼の生き方そのものが、まさに蝶であり蜂だった。

晩年、アリはパーキンソン病を患い、あれほど雄弁だった口は思うように動かなくなる。それでも1996年、震える手で聖火を掲げた彼の姿は、世界中の胸を打った。かつて韻を踏んで敵を挑発した男は、言葉を失ってなお、沈黙のうちに雄弁だった。

華麗な一言の裏には、それを紡いだ相棒の存在と、言葉どおりに生き抜いた一人の男の信念がある。名文句とは、誰が作ったかよりも、誰が本気で生きてみせたかで、伝説になるのかもしれない。
